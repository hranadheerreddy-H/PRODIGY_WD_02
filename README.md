# PRODIGY_WD_02 - Stopwatch Web Application

## Overview
This project is a modern and interactive stopwatch web application developed using HTML, CSS, and JavaScript. The application features a professional glassmorphism UI design with neon glowing effects, responsive layout, and accurate time tracking including milliseconds.

The stopwatch allows users to:
- Start the timer
- Pause the timer
- Reset the timer
- Record lap times

---

# Features

## Stopwatch Functionalities
- Start stopwatch
- Pause stopwatch
- Reset stopwatch
- Record lap times
- Milliseconds support
- Real-time timer updates

## UI/UX Features
- Glassmorphism design
- Neon glowing stopwatch display
- Modern cyber-style interface
- Responsive design for mobile and desktop
- Smooth hover animations
- Stylish lap time records
